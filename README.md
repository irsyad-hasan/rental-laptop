
# Rental Laptop - Laravel Starter

Template sistem rental laptop berdasarkan UI yang diberikan.

Fitur:
- Dashboard Data Laptop
- Data Pelanggan
- Transaksi Rental
- Laporan Rental
- Bootstrap 5 UI dengan gradient header
- Contoh Route, Controller, Blade, Migration

Cara pakai:
1. Buat project Laravel baru.
2. Salin isi folder ini ke project Laravel.
3. Jalankan migration.
4. Sesuaikan model dan logic bisnis.
